Domeinnaam
De URL waarop uw website bereikbaar is
Zonder https:// — bijvoorbeeld: aiventity.nl of www.aiventity.nl
Stap-voor-stap domein instellen
1

Koop een domeinnaam

Registreer uw domeinnaam bij een betrouwbare registrar:
TransIP (aanbevolen voor NL)  ·  Namecheap  ·  Google Domains

2

Stel DNS-records in

Voeg de volgende records toe in het DNS-beheer van uw registrar:

A Record
Naam: @ (of {{ domain_clean or 'uwdomein.nl' }})
Type: A
Waarde: [IP-adres van uw server]
TTL: 3600
CNAME Record (www)
Naam: www
Type: CNAME
Waarde: {{ domain_clean or 'uwdomein.nl' }}
TTL: 3600
3

Installeer Nginx

Installeer Nginx op uw server en gebruik de gegenereerde configuratie rechts. Sla op als /etc/nginx/sites-available/{{ domain_clean or 'aiventity' }}

Zie het codblok rechts voor de volledige configuratie.
4

SSL met Certbot (Let's Encrypt)

Voer de volgende commando's uit op uw server om een gratis SSL-certificaat te installeren:

sudo apt install certbot python3-certbot-nginx -y sudo certbot --nginx -d {{ domain_clean or 'uwdomein.nl' }}{% if domain_clean %} -d www.{{ domain_clean }}{% endif %} sudo systemctl reload nginx # Automatische verlenging testen: sudo certbot renew --dry-run
5

Start de Flask app

Zorg dat de Flask app draait en bereikbaar is op poort 5001. Gebruik een process manager voor betrouwbaarheid:

# Installeer gunicorn pip3 install gunicorn # Start met gunicorn (productie) cd /pad/naar/backend gunicorn -w 4 -b 127.0.0.1:5001 app:app # Of gebruik systemd voor automatisch starten # Maak /etc/systemd/system/aiventity.service aan
